# 🌸 Avika Boutique Website

A modern and elegant boutique website designed using **HTML & CSS**.  
This project represents a real local business concept with a stylish and user-friendly interface.

---

## ✨ Features

- 🎨 Elegant and modern UI design  
- 🖼️ Image-based collection showcase  
- 📱 Responsive layout (basic)  
- 🧭 Navigation bar with sections  
- 🌆 Hero banner with background image  
- 📖 About section with aesthetic design  
- 📞 Contact section
